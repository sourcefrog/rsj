# RSJ cheatsheet

## Verbs

| Verb  | Name   | Meaning                                 |
| ----- | ------ | --------------------------------------- |
| -. x  | not    | 1 when x is 0, otherwise 0              |
| - y   | negate |                                         |
| x - y | minus  |                                         |
| # y   | tally  | the number of items on the leading axis |
