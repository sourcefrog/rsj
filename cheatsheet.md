# RSJ cheatsheet

## Verbs

| Verb  | Name     | Meaning                                                                  |
| ----- | -------- | ------------------------------------------------------------------------ |
| -. y  | not      | 1 when y is 0, otherwise 0                                               |
| - y   | negate   |                                                                          |
| x - y | minus    |                                                                          |
| x + y | plus     |                                                                          |
| # y   | tally    | the number of items on the leading axis                                  |
| $ y   | shape of | a list: empty for an atom, otherwise giving the length of each axis of y |
